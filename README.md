# Aadhaar Anomaly Detection Framework

AI-based anomaly detection framework using Isolation 
Forest algorithm to identify suspicious transactions 
in Aadhaar-linked e-governance systems.

## Paper
This code accompanies the paper:
"AI-Based Anomaly Detection Framework for Privacy 
Protection in Aadhaar-Linked E-Governance Systems"
Submitted to AICYBER 2026, IEEE Conference, Chennai.

## Requirements
pip install numpy pandas scikit-learn

## Usage
python aadhaar_anomaly_detection_code.py

## Results
- Accuracy: 99.84%
- Precision: 99.60%
- Recall: 98.60%
- F1-Score: 99.10%
